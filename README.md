# Youngchampion\_comic\_downloader

A userscript to download manga chapters from youngchampion.jp

## Features

*   Automatically monitors and caches rendered manga pages
*   Provides thumbnail previews of cached pages
*   Supports batch download of cached pages (sorted by page number)
*   Simple control panel with status updates

## Installation

1.  Install a userscript manager (e.g., Tampermonkey, Greasemonkey)
2.  Add this script to your userscript manager

## Usage

1.  Visit a manga chapter page on `https://youngchampion.jp/episodes/*`
2.  A control panel will appear at the top-right corner
3.  Scroll through the manga to load and cache pages
4.  Click "批量下载" (Batch Download) to save all cached pages

## Notes

*   Respect the website's terms of service and copyright laws
*   For personal use only
*   Pages are saved as PNG files with sequential numbering
